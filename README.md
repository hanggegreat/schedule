## 教务系统·排课系统
### 此次开发基于SpringBoot框架2.1.0版本，数据层采用SpringData JPA技术。
### 由于视图会带来开发的不便，本项目将不再建立视图，而是统一将Java实体与数据表进行关联。
---
## 该系统为八个不同角色提供使用，具体角色及权限信息如下：

    </tr>

      <td>5.年级组长</td>

      <td>6.教务员</td>

    </tr>

    </tr>
